Ansible

What is Ansible?

Ansible is an IT automation tool. It can configure systems, deploy software, and orchestrate more advanced IT tasks such as continuous deployments or zero downtime rolling updates.

Architecture

  • Agent-less (no client software required on servers)
  • SSH for communication (secure)

Why Ansible?

  • Low learning curve
  • Doesn't need a "master" server to manage servers
  • No need to install an agent on managed servers

Inventory


                        192.168.6.1

                        [webservers]
                        foo.example.com
                        bar.example.com ansible_ssh_user=deploy

                        [databases]
                        db[1-3].example.com

                        [webservers:vars]
                        document_root=/var/www/app

Playbooks


                        ---
                        - hosts: webservers
                          become: yes
                          tasks:
                            - name: Install PHP
                              apt:
                                pkg: php
                                state: latest
                            - name: Copy php.ini
                              copy:
                                src: php.ini
                                dest: /etc/php/7.1/cli/php.ini
